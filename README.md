
<br>

KODA(한국장기조직기증원)는 부서 간 문의를 보다 신속하고 정확하게 연결하기 위해, 사용자의 질문을 이해하고 관련 부서 정보를 안내하는 챗봇을 도입하였습니다.
이번 프로젝트는 **정확한 부서 응답 제공**, **사용자 편의성 향상**, 그리고 **내부 서비스 대응의 효율성 강화**를 목표로 합니다.
